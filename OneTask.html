Managing multiple tasks can often lead to overwhelm, hindering productivity. To address this, I collaborated with a developer to create "One Task," an app designed to help users focus on a single task at a time, leading to greater efficiency.

PROJECT DURATION:

2 Months - Design and Development

ROLE:

Product Design, Interaction Design, User Research, Usability Testing

DESIGN TOOLS:

Figma, Balsamiq

DESIGN DECISIONS FOR SMARTWATCH:

For the smartwatch integration, we focused on seamless task notifications, optimized UI for smaller screens, and intuitive gestures for quick task management.

DELIVERABLES:

Initial Mockups, High Fidelity Wireframes, Minimal Screens, Persona Chart

Lessons Learned

Through this project, I gained valuable insights into designing for limited screen real estate and prioritizing user interactions for efficient task handling on smartwatches.
